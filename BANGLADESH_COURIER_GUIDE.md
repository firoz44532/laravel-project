## 🎯 **API কনফিগারেশন গাইড**

### **Environment Variables (.env ফাইলে)**

```env
# Steadfast Courier Configuration
STEADFAST_BASE_URL=https://portal.steadfast.com.bd
STEADFAST_API_KEY=your_steadfast_api_key
STEADFAST_SECRET_KEY=your_steadfast_secret_key
STEADFAST_PICKUP_ADDRESS=Your Shop Address, Dhaka

# Pathao Courier Configuration
PATHAO_BASE_URL=https://api-hermes.pathao.com
PATHAO_CLIENT_EMAIL=your_pathao_client_email
PATHAO_CLIENT_PASSWORD=your_pathao_client_password
PATHAO_CLIENT_SECRET=your_pathao_client_secret

# eCourier Configuration
ECOURIER_BASE_URL=https://ecourier.com.bd
ECOURIER_API_KEY=your_ecourier_api_key
ECOURIER_SECRET_KEY=your_ecourier_secret_key
ECOURIER_USER_ID=your_ecourier_user_id
ECOURIER_PICKUP_ADDRESS=Your Shop Address, Dhaka

# RedX Configuration
REDX_BASE_URL=https://redx.com.bd
REDX_API_KEY=your_redx_api_key
REDX_STORE_ID=your_redx_store_id
REDX_PICKUP_ADDRESS=Your Shop Address, Dhaka
```

## 🔧 **ইমপ্লিমেন্টেশন স্টেপস**

### **১. API ক্রেডেনশিয়ালস সংগ্রহ**

#### **Steadfast Courier:**
1. [Steadfast Portal](https://portal.steadfast.com.bd) এ লগইন করুন
2. Settings → API Configuration এ যান
3. API Key এবং Secret Key সংগ্রহ করুন

#### **Pathao Courier:**
1. [Pathao Merchant Portal](https://merchant.pathao.com) এ রেজিস্ট্রেশন করুন
2. API Access এর জন্য আবেদন করুন
3. Client Email, Password, এবং Secret পান

#### **eCourier:**
1. [eCourier Portal](https://ecourier.com.bd) এ একাউন্ট খুলুন
2. API Integration এর জন্য যোগাযোগ করুন
3. API Key, Secret Key, এবং User ID পান

#### **RedX:**
1. [RedX Merchant](https://redx.com.bd/merchant) এ সাইন আপ করুন
2. Store তৈরি করুন এবং API Access নিন
3. API Key এবং Store ID পান

### **২. ডাটাবেস মাইগ্রেশন**

```bash
php artisan migrate
```

### **৩. কুরিয়ার সার্ভিস কনফিগারেশন**

```php
// config/services.php এ কুরিয়ার কনফিগারেশন যোগ করুন
// CourierFactory.php ব্যবহার করে ডায়নামিকভাবে কুরিয়ার সিলেক্ট করুন
```

## 📝 **নোট:**

এই গাইডটি বাংলাদেশের সবচেয়ে জনপ্রিয় কুরিয়ার সার্ভিসগুলোর সম্পূর্ণ তথ্য দেয়। API ইন্টিগ্রেশনের জন্য সরাসরি কুরিয়ার কোম্পানির সাথে যোগাযোগ করে API ক্রেডেনশিয়ালস সংগ্রহ করুন।

**মনে রাখবেন:** সব কুরিয়ারের API একই রকম কাজ করে না। কিছু কুরিয়ারের লিমিটেড API সাপোর্ট আছে, তাই ইন্টিগ্রেশনের আগে API ডকুমেন্টেশন ভালোভাবে পড়ুন।
